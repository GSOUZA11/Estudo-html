# HTML - Linguagem de Marcação de Hipertexto < p > Pricipais elementos especiais da marcação HTML ###cabeça < p > O elemento HTML < head > informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos. ###título < p > O elemento HTML título define o título do documento, exibido na barra de título de um navegador ou na aba da página. ###corpo < p > Representa o conteúdo de um documento HTML ###cabeçalho < p > Representa um grupo de suporte introdutório ou navegacional, pode conter alguns elementos de cabeçalho, mas também outros elementos como logotipo, seções de cabeçalho, formulário de pesquisa e outros. ###rodapé < p > Representa um rodapé, contendo informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados ###artigo < p > Representa uma composição independente em um documento, página, aplicação, ou site, ou que é destinado a ser distribuído de forma independente ou reutilizável, por exemplo, em sindicação. ###seção < p > Representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existe um elemento semântico mais específico para representá-lo. ###p < p > Representa um parágrafo ###div < p > É um container genérico para conteúdo de fluxo, que de certa forma não representa nada. Ele pode ser usado para agrupar elementos para fins de estilos (usando classe ou id), ou porque eles desenvolveram valores de atributos, como lang. ###período < p > É um contêiner genérico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id ), ou para compartilhar valores de atributos como idioma. ###img < p > Representa a inserção de imagem no documento, sendo implementado também pelo HTML5 para uma melhor experiência com o elemento < figure > e < figcaption > . ###a parte, de lado < p > Representa uma seção de uma página que consiste em um conteúdo tangencialmente relacionado ao conteúdo de seu ambiente, que poderia ser considerado separado do conteúdo. ###áudio < p > É utilizado para embutir conteúdo de som em um documento HTML ou XHTML. ###tela de pintura < p > pode ser usado para desenhar gráficos utilizando scripts (geralmente JavaScript). ###datalist < p > O elemento datalist contém um conjunto de elementos < option > que representam as opções possíveis para o valor de outros controles. ###detalhes < p > O elemento é usado como uma ferramenta de onde o usuário irá obter informações adicionais. ###Embutir < p > O elemento incorpora conteúdo externo no ponto especificado no documento ###nav < p > O elemento representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação. ###saída < p > é um elemento no qual um site ou aplicativo pode injetar os resultados de um calculador ou o resultado de uma ação do usuário. ###progresso < p > É usado para visualizar o progresso de uma tarefa. Embora as especificidades de como é mostrado fiquem a carga do desenvolvedor, normalmente, é mostrado como uma barra de progresso. ###vídeo < p > É utilizado para incorporar conteúdo de vídeo em um documento HTML ou XHTML. ###ul < p > O elemento representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordem numérica e as suas posições, nessa lista, são irrelevantes. ###ol < p > O elemento representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou símbolos simples. ###li < p > É usado para representar um item que faz parte de uma lista. Este item deve estar contido em um elemento pai: uma lista ordenada ( < ol > ), uma lista desordenada ( < ul > ), ou um menu ( < menu > (en-US)) e representa uma única entidade dessa lista. ## Referência - [HTML: Linguagem de Marcação de Hipertexto](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
